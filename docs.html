Project Overview

Welcome to the Advanced 3D Expense Tracker — a next-generation web-based personal finance application powered by Streamlit and Three.js.

Transform your browser into a futuristic financial command center with 3D animated cyberpunk UI, interactive 3D charts, Three.js particle backgrounds, AI financial insights, and a multi-user system with secure authentication.

Quick Note: This application runs entirely on your local machine. All data is stored in a local SQLite database — no data leaves your computer.

Prerequisites

Before installing, ensure you have the following:

  • Python 3.11 or higher
  • pip (Python package manager)
  • Git (for cloning the repository)
Check Python Version 
python --version
# or
python3 --version

Installation

Step 1: Clone the Repository

Git Clone 
git clone https://github.com/issu321/Advanced-CLI-Expense-Tracker.git
cd Advanced-CLI-Expense-Tracker

Step 2: Install Dependencies

Choose one of the following methods based on your operating system:

Windows Setup

Option 1: Automatic Installation (Recommended)

Simply run the provided batch file:

Windows Auto-Install 
install.bat

Option 2: Manual Installation

Manual Windows Setup 
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py

Then open your browser at: http://localhost:8501

Linux Setup

Option 1: Automatic Installation (Recommended)

Linux Auto-Install 
chmod +x install.sh
./install.sh

Option 2: Manual Installation

Manual Linux Setup 
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
streamlit run app.py

Then open your browser at: http://localhost:8501

Manual Setup

If the automatic installers don't work, follow these manual steps:

1

Create Virtual Environment

python -m venv venv
2

Activate Virtual Environment

Windows: venv\Scripts\activate

Linux/Mac: source venv/bin/activate

3

Install Dependencies

pip install -r requirements.txt
4

Run the Application

streamlit run app.py

Database Schema

The application uses SQLite3 (expense_data.db) with the following schema:

Table Purpose
users Stores user accounts with SHA-256 hashed passwords
expenses All expense transactions per user
income All income records per user
budgets Monthly budget allocations by category
savings_goals Savings targets with progress tracking
recurring_expenses Automated recurring bill/subscription data

Data Security: All passwords are hashed using SHA-256. Each user has fully isolated financial records. No data leaves your local machine.

Contributing Guide

We welcome contributions! Follow these steps:

1

Fork the Repository

Click the Fork button on the GitHub repo.

2

Clone Your Fork

git clone https://github.com/your-username/Advanced-CLI-Expense-Tracker.git
3

Create a Feature Branch

git checkout -b feature/amazing-feature
4

Commit Your Changes

git commit -m 'Add amazing feature'
5

Push and Open PR

git push origin feature/amazing-feature

Then open a Pull Request on GitHub.

Coding Standards: Follow PEP 8 style guidelines. Keep functions focused and lightweight. Add docstrings for new features. Test on both Windows and Linux.
